# GraphQL POC - Spring Boot

This project is a simple Proof of Concept (POC) demonstrating the use of GraphQL with Spring Boot, JPA, and H2 database.

## 🚀 Technologies

- Java 17+
- Spring Boot
- Spring GraphQL
- Spring Data JPA
- H2 Database
- Lombok

## 📌 Features

- Create categories
- Create products
- Associate products with categories
- Query products with nested category data
