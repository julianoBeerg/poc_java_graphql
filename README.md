A simple Proof of Concept (POC) demonstrating how to build a GraphQL API using Spring Boot, JPA, and H2 database.

---

## 📖 Overview

This project showcases a basic GraphQL implementation with:

- Query and Mutation operations
- Entity relationships (Product ↔ Category)
- In-memory database for quick testing
- Clean and simple architecture for learning purposes

---

## 🧱 Tech Stack

- **Java 17+**
- **Spring Boot**
- **Spring GraphQL**
- **Spring Data JPA**
- **H2 Database**
- **Lombok**

---

## 📌 Features

✔️ Create categories  
✔️ Create products  
✔️ Link products to categories  
✔️ Fetch products with nested category data
